System

Overview

The system class contains methods for getting OS standard file system icons, preferences, settings and information.

clipboardSet(data, datatype)
Description

Sets the clipboards data and adds the specified type to it. The type is case sensitive.

Definition

system.clipboardSet(data, datatype)
Parameters
NameTypeDescription
data string The data to set on the clipboard (Buffer data or string usually).
datatype string The type to search for data, 'text', 'image', 'rtf', 'html', 'video', 'audio'
Returns
TypeDescription
Object An object with one function, "release" which releases the data back to memory once it should no longer be available in the clipboard. IMPORTANT: Data placed in the clipboard is not automatically memory managed on some platforms, ensure you release data before setting new data on the clipboard as a precautionary measure. Note if an error occurs this operation returns false.

sendKey(input)
Description

Sends a key down and key up event through the OS, note the currently focused application, window and control will receive this event (and may not be the running application). This is mainly used for testing and should be avoided in production applications.

Definition

system.sendKey(input)
Parameters
NameTypeDescription
input string The character to send as a key up and key down event. Note this must be one single character e.g., 'a', 'b', or 'A', or 'B'. For a carrage return use 'ENTER' or 'RETURN', for shift use 'SHIFT', etc.
